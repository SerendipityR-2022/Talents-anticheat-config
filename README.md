### 🔧如何安装🔧

#### 1.查看[这里](https://github.com/TalentsDX/Talents-anticheat-config/releases)下载你想要的反作弊搭配，获取里面的yml文件。
####  2.替换掉原先反作弊的yml文件。
####  3.重启你的服务器。
#### 4.享受反作弊！
